

# 🎨 Color Detection Project

## 📌 Overview

The Color Detection Project is an interactive computer vision application developed using Python and OpenCV. It allows users to identify colors in an image by selecting pixels and also provides HSV-based color filtering using adjustable trackbars.

This project demonstrates image processing, color space conversion, mouse events, and HSV thresholding concepts.

---

## 🚀 Features

* Interactive color detection using mouse double-click.
* Displays the nearest color name.
* Shows RGB and HEX color values.
* HSV threshold adjustment using trackbars.
* Real-time color masking.
* Save screenshots of the output.
* Simple and user-friendly interface.

---

## 🎯 How to Use

1. Run the project.
2. Double-click on any pixel in the image.
3. The application displays:

   * Color Name
   * RGB Values
   * HEX Code
4. Use the HSV trackbars to isolate different colors.
5. Press **S** to save a screenshot.
6. Press **Q** or **ESC** to exit.
